备注

1

地平线J5 AI芯片工具链用户手册

J5_Ai_Toolchain_User_Guide

本手册全面介绍芯片工具链产品，帮助开发者了解：环境部署、两种模型处理方案、模型部署、应用开发、自定义算子开发等工具链开发全流程。此外，还展示了一系列算法模型的Benchmark性能数据。

2

J5 AI芯片工具链工具手册

J5_AI_Toolchain_Tool_Docs

详细介绍芯片工具链提供的: PTQ浮点模型转换工具链和Runtime应用开发工具包。

3

PTQ模型转换示例手册

J5_AI_Toolchain_Sample_Docs

介绍horizon_model_convert_sample模型转换示例包及其使用方法。提供了使用浮点模型转换工具链快速完成浮点模型转换定点模型的示例，同时包括单张推理和精度验证示例。

4

算子支持与约束列表

supported_op_list_and_restrictions

详细介绍地平线平台支持的Caffe和ONNX算子及其约束条件。

5

BPU SDK API Doc

BPU_SDK_API_DOC

介绍地平线runtime SDK的API、数据、结构体、排布和对齐规则。

6

DSP RPC API Doc

DSP_RPC_API_DOC

介绍地平线dsp rpc的API。

7

Horizon Algorithm Toolkit

HAT

详细介绍地平线基于PyTorch开发的HAT (Horizon Algorithm Toolkit)深度学习训练工具。

8

基础示例包使用说明

J5_BPU_Runtime_Sample_Docs

介绍 horizon_runtime_sample 嵌入式开发基础示例包及其使用方法。提供了使用API完成单张图片推理、含自定义算子模型的实现和推理，多模型批量推理等示例，帮助开发者快速完成模型部署。

9

AI-Benchmark使用说明

J5_AI_Benchmark_Docs

介绍 ai_benchmark 嵌入式应用开发示例包及其使用方法。开发者使用该示例包可以体验模型部署和模型性能评估相关的应用开发。示例脚本包括性能评测和精度评测两部分。
