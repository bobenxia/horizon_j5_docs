Horizon Algorithm Toolkit (海图)
================================

Horizon Algorithm Toolkit是地平线提供基于Pytorch的深度学习训练
工具。


HAT算法包是基于Pytorch的接口开发的一套高效且用户友好的AI算法工具。同时它还可以提供包含分类，检测，分割等常见的图像
任务的SOTA(state-of-the-art)深度学习模型。
